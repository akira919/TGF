# 意外事件

我是志翔，一個二十四歲年輕有為的科學家。

對于周圍的美眉，我總是感到非常害羞，所以直到現在都沒有拍過拖。

我們正嘗試研究出一種治療任何癌癥的新藥。

過後，等她回來的時候，無聊的我就拿那新藥來玩，意想不到的是發生了爆炸。

接著我知道我已經變成透明了。

我問我自己「我死了嗎？

」答案是連我自己也不知道。

我俯頭一看，地上只有燒傷標記。

忽然間听見一陣敲門聲，隨即被打開了，曉穎匆匆忙忙地跑進來。

 她大叫「翔，你在哪里？

我听到這里有爆發聲。

」 我道「曉穎我就在這里！

」但她好像听不到我的聲音。

 看到地下的燒傷標記後，她開始哭了，她以為我已經意外喪命了。

我應該讓她知道我還未死的，就在她的身邊而已。

但我應該怎麼辦呢？

我試著看可不可以像傳說的鬼那樣任意附在他人身體上，以便留下信息讓她可以知道我在哪里。

我慢慢走近她，然後把自己的手放在她的小手上。

成功了！

我開始在桌上的白紙寫字。

你能听得到我的話，我有一個好主意。

你可以附在我的身體上來做你自己的解藥。

】 這時，我忽然感到很緊張，慢慢的走向她。

這是我的第一次附身，而且還是附在曉穎的身體上。

那乳溝似乎要呼之欲出。

現在我確實已是曉穎了。

然後我看著寧靜的實驗室，我知道自己不應該白白浪費時間和曉穎的苦心。

我立刻埋頭開始嘗試做實驗找解藥。

 意外事件一個星期後，我發現每一次附身時曉穎都會昏迷，完全不知道我所做過的任何事情。

我們通過寫筆記來保持聯絡。

在這段時期，曉穎她都在我的公寓留宿。

這樣我可以專心在自己的實驗室做試驗。

接著我又在筆記寫【有事想要請你幫忙，可以嗎？

這可能是我一生人的唯一的機會。
